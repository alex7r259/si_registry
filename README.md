# Веб-приложение учета средств измерений (СИ)

## Стек
- **Backend:** Python + Flask
- **База данных:** SQLite
- **Frontend:** HTML + Bootstrap 5 + DataTables
- **ORM:** SQLAlchemy

## Возможности
- Таблица перечня средств измерений.
- Поиск по любой колонке.
- Добавление новых СИ.
- Редактирование существующих СИ.
- Удаление СИ.
- Пометка «В поверке».
- Автоматическая подсветка сроков поверки:
  - Просрочено → красный.
  - Остался месяц → желтый.
- Стандартная сортировка по дате (ближайшая дата поверки сверху).
- Хранение данных в SQLite.
- Адаптивный интерфейс.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска откройте: `http://127.0.0.1:5000`
